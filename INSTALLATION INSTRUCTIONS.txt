# Node.js required
1 - Download Node.js from https://nodejs.org/en/download/

2 - Open the index.html file folder, open a command prompt and type the following commands: 

    npm init 
    npm install "@azure/cognitiveservices-computervision"
    npm install "@azure/ms-rest-js"
    npm install "bootstrap"

3 - Open index.html in your favorite browser, as long as the favorite browser is not Internet Explorer.
